All protected endpoints require authentication via Authorization: Bearer <API_KEY>

📁 Files Endpoints

GET /api/files List all files
GET /api/files/stats Get statistics
GET /api/files/:id Get file details
POST /api/files/upload Upload a file
DELETE /api/files/:id Delete a file

🔑 Tokens Endpoints

GET /api/tokens List all tokens
POST /api/tokens Create access token
DELETE /api/tokens/:id Revoke token

📥 Download (Public)

GET /download/:token Download file with token
HEAD /download/:token Check file availability

💻 Quick Start

# Upload a file
curl -X POST http://localhost:8899/api/files/upload \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -F "file=@image.jpg"

# Create download token
curl -X POST http://localhost:8899/api/tokens \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"fileId": "FILE_ID", "expiresInHours": 24}'

# Download (no auth needed)
curl -O http://localhost:8899/download/TOKEN
